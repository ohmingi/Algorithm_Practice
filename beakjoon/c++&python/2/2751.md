## 백준 2751번 수 정렬하기 2 (C++)

이번에 풀어본 문제는 **백준 2751번 수 정렬하기 2**이다.

문제는 아래 링크에서 확인할 수 있다.  
[백준 2751 문제 바로가기](https://www.acmicpc.net/problem/2751)

### 💻 코드

```
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main(){
    int n;
    cin >> n;

    vector<int> v;

    while(n--){
        int a;
        cin >> a;
        v.push_back(a);
    }

    sort(v.begin(), v.end());

    for(int a : v){
        cout << a << "\n";
    }
}
```

### 🔍 로직 설명

sort함수가 무슨 시간복잡도를 가지는지만 알면 쉽게 푸는 문제   
sort함수는 하이브리드하게 정렬 방식을 바꾸기에 데이터의 크기에 따라 시간복잡도가 달라지는데 

-   **평균**: O(n log n)
-   **최악**: O(n log n)
-   **최선**: O(n) (거의 정렬된 경우 삽입정렬이 빠르게 작동)

그렇기에 n의 최대값이 1,000,000이라도 1초안에 해결된다

### ✍️ 느낀 점

sort함수의 시간복잡도를 잘 기억해놓자
