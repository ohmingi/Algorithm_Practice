<p data-ke-size="size16">문제는 아래 링크에서 확인할 수 있다.<br /><a href="https://www.acmicpc.net/problem/2004" target="_blank" rel="noopener noreferrer">백준 2004 문제 바로가기</a></p>
<h3 id="💻-코드" data-ke-size="size23">💻 코드</h3>
<pre class="arduino"><code>#include &lt;iostream&gt;
using namespace std;

long long count_p(long long n, int p) {
    long long cnt = 0;
        while (n &gt; 0) {
        cnt += n / p;
        n /= p;
    }
    return cnt;
}

int main() {
    long long n, m;
    cin &gt;&gt; n &gt;&gt; m;

    long long two = count_p(n, 2) - count_p(m, 2) - count_p(n-m, 2);
    long long five = count_p(n, 5) - count_p(m, 5) - count_p(n-m, 5);

    int a = min(two, five);

    cout &lt;&lt; a;
}
</code></pre>
<h3 id="🔍-로직-설명" data-ke-size="size23">🔍 로직 설명</h3>
<p data-ke-size="size16">처음엔 ()가 뭘 의미했는지 기억이 안나서 1차 당황했다.<br />조합이라는 것을 알게되었고, 풀려고 하는데 당연히 모든 곱을 다 게산하면 컴퓨터가 터질것이라고 알고 있었다.<br />그래서 고민을 했는데, 음 0을 만드는거니까 10을 찾자 하고 찾으려고함수를 만들려고 했는데 팩토리얼값의 2와5의 개수를 어떻게 구해야 할지 어려웠다.</p>
<p data-ke-size="size16">기술의 힘을 빌려 함수를 짰는데, 2의 개수를 찾을때는 먼저 12!같은 경우는 12 / 2를 통해 2의 배수가 몇개있는지 구한다<br />이때 4같은 경우는 2가 두번있는것이니까 4의배수도 몇개인지 구한다. 같은경우로 8은 2가 3개니까 8의 배수도 구한다<br />쭉쭉 0이 될때까지 구하면 2의 개수를 구할 수있고, 5도 마찬가지로 하면된다</p>
<p data-ke-size="size16">분수 형태니까 12c3이라면 9!와 3!값도 구해서 빼주면 2의 개수와 5의 개수가 나오고 그중 작은 수의 개수만큼 출력하면 0의 개수가 나온다</p>
