## 백준 2667번 단지번호붙이기 (C++)

이번에 풀어본 문제는 **백준 2667번 단지번호붙이기**이다.

문제는 아래 링크에서 확인할 수 있다.  
[백준 2667 문제 바로가기](https://www.acmicpc.net/problem/2667)

### 💻 코드

```
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int n;
int graph[26][26];
bool visited[26][26];
int c = 0;

int dx[4]={0,0,1,-1};
int dy[4]={1,-1,0,0};

bool check(int x, int y) {
  return x >= 0 && y >= 0 && x < n && y < n;
}

void dfs(int x, int y) {
  c++;
  visited[x][y] = true;

  for (int i = 0; i < 4; i++) {
    int nx = x + dx[i];
    int ny = y + dy[i];
    if (check(nx, ny) && !visited[nx][ny] && graph[nx][ny] == 1) {
      dfs(nx, ny);
    }
  }
}

int main() {
  cin >> n;

  for (int i = 0; i < n; i++) {
    string s;
    cin >> s;
    for (int j = 0; j < n; j++) {
      graph[i][j] = s[j] - '0';
    }
  }

  int count = 0;
  vector<int> v;

  for (int i = 0; i < n; i++) {
    for (int j = 0; j < n; j++) {
      if (graph[i][j] == 1 && visited[i][j] == false) {
        c = 0;
        count++;
        dfs(i, j);
        v.push_back(c);
      }
    }
  }

  sort(v.begin(), v.end());

  cout << count << "\n";
  for (int a : v) {
    cout << a << "\n";
  }
}
```

### 🔍 로직 설명

단순 dfs돌려서 완전탐색하는 문제  
처음에 입력값이 int깂으로 들어오는 줄 알았는데, string값이여서 살짝 헤멨다.  
string각 인덱스를 int로 변환해서 저장하면 문제없다

시간복잡도는 for문이 n^2반복하고 dfs도 최대 n^2반복이고 visited로 인해 for문이 최대 n^2 돌았을 경우 dfs는 최대 한번 돌아가기에 총 시간복잡도는 O(n^2)이다

### ✍️ 느낀 점

완전탐색문제 풀때 입력값이 공백과 int로 주어진것인지, string으로 주어진것인지 잘 확인하고 문제를 풀자.
