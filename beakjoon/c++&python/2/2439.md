<h2 data-ke-size="size26">백준 2493번 탑 (C++)</h2>
<p data-ke-size="size16">이번에 풀어본 문제는 스택(Stack) 자료구조를 활용하는 대표적인 문제인 <b>백준 2493번 탑</b>입니다. 왼쪽으로 레이저를 발사했을 때 어떤 탑에서 수신하는지를 찾는 문제로, 스택을 이용하면 O(N)의 시간 복잡도로 효율적으로 해결할 수 있습니다.</p>
<p data-ke-size="size16">문제는 아래 링크에서 확인할 수 있습니다.<br /><a href="https://www.acmicpc.net/problem/2493" target="_blank" rel="noopener noreferrer">백준 2493번 문제 바로가기</a></p>
<h3 id="💻-코드" data-ke-size="size23">💻 코드</h3>
<pre class="cpp" data-ke-language="cpp"><code>#include &lt;iostream&gt;
#include &lt;stack&gt;
#include &lt;vector&gt;

using namespace std;

int main(){
int n;
cin &gt;&gt; n;

    stack&lt;pair&lt;int,int&gt;&gt; st;
    vector&lt;int&gt; v;
    v.push_back(0);

    for(int i = 1; i &lt;= n; i++){
        int a;
        cin &gt;&gt; a;
        if(st.empty()){
            st.push({a,i});
        }
        else if(a &gt; st.top().first){
            while(!st.empty() &amp;&amp; st.top().first &lt; a){
                st.pop();
            }
            if(st.empty()) v.push_back(0);
            else{
                v.push_back(st.top().second);
            }
            st.push({a,i});
        }
        else{
            v.push_back(st.top().second);
            st.push({a,i});
        }
    }

    for(int a : v){
        cout &lt;&lt; a &lt;&lt; " ";
    }
}</code></pre>
<h3 id="🔍-로직-설명" data-ke-size="size23">🔍 로직 설명</h3>
<p data-ke-size="size16">일단 단순하게 오른쪽 끝값부터 왼쪽 모든값을 순회하는 형식으로 짜면 n^2의 시간복잡도를 가지기 때문에 당연히 아니었고, 그렇다면 dp나 스택의 관점에서 접근해보려고 노력했었다.<br />이중 dp를 해서 값이랑 index저장해서 뒤쪽에서 혹은 앞쪽에서부터 해볼까라고 생각했었는데, 어떻게 해서도 의미가 없었다.<br />뒤에서 탐색하는 방법은 완전탐색이랑 다를게 없었고, 앞에서부터 탐색은 이전의 모든 값들을 저장하면서 가야하기에 dp문제가 아니라고 느꼈다. 그렇다면 보통 큐나 스택으로 구할거같은 문제인데, 이전으 값들을 저장해놓고 다음값을 탐색하기에 스택문제라고 생각했다.</p>
<p data-ke-size="size16">그래서 각경우들을 나누어서 스택에 index랑 값을 저장하고 풀었다.<br />다른경우들은 크게 생각할일 없는데 지금 입력받은 값이 스택 top보다 크다면 pop을 하면서 입력받은 값보다 큰값을 찾는 코드를 조금 신중하게 짜야한다 ex) 스택이 비어있을때, 큰 값이 없을때 등등</p>
<p data-ke-size="size16">그 이외에는 크게 고민할 부분은 없는 문제였다</p>
