<h2 data-ke-size="size26">백준 2609번 최대공약수와 최소공배수 (C++)</h2>
<p data-ke-size="size16">이번에 풀어본 문제는 수학의 기본 개념인 최대공약수(GCD)와 최소공배수(LCM)를 구하는 <b>백준 2609번 문제</b>입니다. 두 수를 입력받아 유클리드 호제법을 이용하여 간단하게 해결할 수 있었습니다.</p>
<p data-ke-size="size16">문제는 아래 링크에서 확인할 수 있습니다.<br /><a href="https://www.acmicpc.net/problem/2609" target="_blank" rel="noopener noreferrer">백준 2609번 문제 바로가기</a></p>
<h3 id="💻-코드" data-ke-size="size23">💻 코드</h3>
<pre class="cpp"><code>#include &lt;iostream&gt;

using namespace std;

int main(){
&nbsp; &nbsp; int a,b;
&nbsp; &nbsp; cin &gt;&gt; a &gt;&gt; b;

&nbsp; &nbsp; int aa = a;
&nbsp; &nbsp; int bb = b;

&nbsp; &nbsp; while(b != 0){
&nbsp; &nbsp; &nbsp; &nbsp; int c = b;
&nbsp; &nbsp; &nbsp; &nbsp; b = a % b;
&nbsp; &nbsp; &nbsp; &nbsp; a = c;
&nbsp; &nbsp; }

&nbsp; &nbsp; cout &lt;&lt; a &lt;&lt; "\n" &lt;&lt; aa*bb / a;
}
</code></pre>
<h3 id="🔍-로직-설명" data-ke-size="size23">🔍 로직 설명</h3>
<p data-ke-size="size16">흔히 말하는 유클리드 호제법으로 풀 수 있는 문제다 a와 b를 모듈러 연산해서 나머지가 0일때가지 돌리고 그때 a % b라면 a값이 최대공약수가 된다.</p>
<p data-ke-size="size16">여기까진 아는거라서 쭉하고 최소공배수를 하려는데 어라 어떻게 하더라라고 생각이 들었다. 이래서 수학구현 문제는 쉬운문제도<br />풀어줘야한다.</p>
<p data-ke-size="size16">정답은 a와b를 곱하고 구한 최대공약수로 나눠주면 최소공배수가 나온다</p>
<p data-ke-size="size16">증명과 풀이는 위대한 수학자분들께서 해주셨을테니 구글을 참고..ㅎ</p>
<h3 id="✍️-느낀-점" data-ke-size="size23">✍️ 느낀 점</h3>
<p data-ke-size="size16">최대공약수 = 유클리드 호제법<br />최소공배수 = a*b / 최대공약수</p>
<p data-ke-size="size16">쉬운 문제도 열심히 풀자 놓치는 수학적 알고리즘이 있을지도 모른다.</p>
